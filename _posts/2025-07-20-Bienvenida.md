# Bienvenido al Blog de Route Manager

Este es un blog en el que vamos a mostrar todo el proceso de análisis, diseño y desarrollo del proyecto para
Estructuras de Datos 2025-1 del grupo **La Corporacíon**

# Introdución

Route Manager SaaS es una plataforma web que ofrece a emprendedores y pymes un servicio de gestión de rutas y
mensajería (on-demand o programada) sin necesidad de desarrollar su propia infraestructura. Nuestro objetivo
es brindarles control absoluto sobre cada entrega o recolección, ya sea que usen sus propios vehículos y
conductores, contraten servicios externos o hagan ambas cosas. La solución se apoya en: 

- Backend en Java (con módulos opcionales en Node.js/JavaScript para el front). 

- Autenticación y roles (Cliente, Conductor, Gestor de Rutas), con contraseñas encriptadas. 

- Automatización vía WhatsApp, mediante un árbol de decisión que permite confirmar, reprogramar o cancelar
servicios. 

- Estructuras de datos (listas enlazadas, montículos, tablas hash, árboles) para modelar rutas, colas de
servicios y flujos de interacción. 

- Desarrollo ágil con Scrum, y definición de historias de usuario bajo criterios SMART e INVEST.
